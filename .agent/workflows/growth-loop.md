# Workflow: The Growth Loop

1. **Strategy**: Call `ANALYTICS_AGENT.md` and define your core value prop.
2. **Schema**: Direct the agent to generate the Event Tracking Schema for your tech stack.
3. **Memory**: Call `MEMORY_PROMPT.md` to persist the schema.
4. **Ticketing**: Call `PROJECT_MANAGEMENT_AGENT.md` to create tickets for the analytics integration.
5. **Execution**: Follow the **Implementation Loop** to code and verify the tracking.
6. **Analysis**: Once live, return to `ANALYTICS_AGENT.md` to analyze funnels and identify the "Aha! Moment."
