# Workflow: Deployment & Memory

1. **Deploy**: Call `GCP_DEPLOYMENT_AGENT.md` or `AZURE_DEPLOYMENT_AGENT.md`.
2. **Verify**: Perform the manual and automated health checks provided by the agent.
3. **Persist**: Call `MEMORY_PROMPT.md` to archive the session state and update the Squad Dashboard.
