# Workflow: PRD to Blueprint

1. **Architecture**: Call the `SW_ARCHITECTURE_AGENT.md` and provide the PRD.
2. **Memory**: Call `MEMORY_PROMPT.md` to persist the technical architecture.
3. **Design**: Call `UI_DESIGN_AGENT.md` to establish visual identity.
4. **Memory**: Call `MEMORY_PROMPT.md` to persist visual tokens.
5. **Feature Spec**: Call `UI_FEATURE_DESIGN_AGENT.md` to create functional blueprints.
6. **Memory**: Call `MEMORY_PROMPT.md` to finalize blueprints for execution.
